הסדרן שעובד בשבילך

מערכת חכמה שסורקת את הקבוצות בוואטסאפ אצלך בטלפון, מסננת לפי העיר שאתה בוחר, ושולחת אליך למחשב רק נסיעות/משלוחים רלוונטיים — בלי רעש, בלי גלילה ובלי לבזבז זמן.

למי זה מתאים?

  • 🚕נהג מונית — רוצה לקבל רק הודעות שמתאימות לעיר/אזור שלך.
  • 🚌נהג הסעות — רוצה לזהות נסיעות מתאימות במהירות ולסגור עבודה.
  • 🏍️שליחים — רוצה לראות רק משלוחים רלוונטיים בלי לטבוע בהודעות.

איך מתחילים לעבוד? (צעד-צעד)

  1. פתח את WhatsApp Web במחשב (דרך התוכנה).
  2. במקום שבו רשום START — עכשיו רשום: סרוק ב-QR את הוואטסאפ בטלפון שלך
  3. הגדר את מספר הטלפון שלך בתוכנה (כמו שמופיע בהגדרות).
  4. הוסף את קבוצות היעד שלך לקטגוריה "מועדפים" בוואטסאפ.
  5. במקום שבו רשום / שם — רשום: את/אתה
  6. בחר עיר/אזור עבודה → והמערכת תציג לך רק הודעות חדשות ורלוונטיות.

✅ הכל עובד בצורה פשוטה: אתה קובע את העיר — והסדרן מסנן עבורך את ההודעות החדשות.

פקודות שימוש מהירות בוואטסאפ:

  • פנוי באשדוד – מציין שאתה פנוי בעיר אשדוד.
  • פנוי בתל אביב+ – פנוי ומחפש גם בערים קרובות.
  • תפוס – עוצר שליחת נסיעות זמנית.
  • עצור – עוצר את המערכת עד הודעה חדשה.
  • רענון – מרענן את הסינון והמצב.

הפקודות נשלחות בוואטסאפ בצורה חופשית, בדיוק כמו הודעה רגילה.
